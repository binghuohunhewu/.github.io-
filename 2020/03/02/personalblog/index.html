针对产出者的自搭博客选择指南

杂七杂八
  • 有感而发
  • 自搭博客

以下内容由（既没学过代码也没有程序基础，只是自己搭过博客的）博主综合整理写成，鉴于博主确实是个真·外行，以下内容不可避免地会出现疏漏和错误，有些专用名词也用得不太准确，如果有行家看出问题，还请多多指点，谢谢。

特别感谢：在本文整理过程中出谋划策的大佬们

为什么要自己搭博客？

明明有那么多的在线博客，注册一下就能用了，还是傻瓜式后台，为什么非要费时费力地自己搭博客呢？

有段时间，产出太太们都喜欢用废柴兔——然后它被墙了。
然后有段时间，大家都在用博客大巴——然后博客大巴关站了。
曾在写手们中间风靡一时、被用来存放同人文产出的不老歌，因为政策问题，不得不谢绝了所有非日记类博文。
百度空间，已经消失在历史长河里了。
Pixiv也可以放小说的哦？……墙了。
lofter，博文改了八百圈都发不出去搞得满脑袋火，还不知道到底哪个是敏感词。
晋江，就不说脖子以下和各种奇妙操作了，蹲□□子、影□□之类的打码时有出现，影响阅读顺畅性。
AO3？我不就是因为这事才来写的这个整理吗（血流满面.jpg）

结论：有些网站不靠谱，有些网站还算靠谱但是打不过老大哥，与其在各个网站中间来回流浪到处搬家，不如给自己搞个可以安心的窝，这样……哪怕还要继续流浪也有底气了是吧囧

自搭博客需要做什么准备？


时间和精力以及耐心

与在线博客网站不同，自搭博客是要从零开始的。
即使网络上有很多傻瓜式教程，你一步一步照抄下来就能成功，也可能会面对“为什么会出bug？！问题在哪？！我写的代码跟教程一模一样啊？！……哦我把代码的英文冒号写成中文冒号了”这样的令人无语的问题，请做好心理准备。

准备三大要件

一个能被大家看到的自搭博客通常由三个要件组成：
【能进入这个博客的办法】、【搭建起博客的系统】、【存放系统和你的博文的网络空间】。

域名、IP

想找到一幢房子，需要知道房子的经纬度，或者是写着所在街道和门牌号的地址。
IP由数字串组成（比如123.456.789.10这样的格式），如同经纬度，精确地指出了你的博客所在的位置，可以用ip进入博客（我记得早年A站只用ip用了好长时间？），但是因为是纯数字，可能不太方便记忆。
有没有像街道和门牌号一样，相对较方便的记法呢？这个时候需要域名。起一个比较记得住的域名，把域名和ip进行绑定和对应，就可以通过域名进入站点了。一个站点可能有很多个域名，也可能有主次或者主要和备用之分，但是指向的都一样是这个站点。
举例：AO3的主域名被墙了，但是用副域名仍然可以进入首页，虽然域名不一样，进入的都是这一个站。
结论：为了能让其他人访问你的博客，你需要一个IP地址，也可以给这个ip绑定一个（或者几个）域名。

博客框架

不同的博客系统，使用方法也不尽相同。
最常见的应该是wordpress？用这个的人很多，各种大佬开发的小插件之类也不少，可以想怎么折腾就怎么折腾。
另外我现在这个博客用的是hexo，我个人感觉用得还挺开心的，但是确实比wp麻烦23333
mooncellwiki群里的大佬们提到的其他博客系统：Typecho，jekyll，vblog。

空间（服务器）

博文和博客系统需要存放在服务器空间里。
根据使用的博客系统的不同，有些博客系统需要你在特定的网站注册一些帐号，另外一些博客系统则需要你拥有服务器空间。
上文所说的hexo、jekyll和vblog是基于github网站的系统，需要注册github帐号。wordpress系统需要服务器空间。

根据自己的需求作出选择

域名一定要有吗？怎么获得？

  • 如前段所述，即使没有域名，有ip（或者二级域名）也是可以进入博客的，所以“搞个新域名”不是必须的，请大家根据自身情况（钱包充盈度）自行选择。
  • 如果你很想搞一个“高大上的只属于我的特别的域名”，你需要去贩卖域名的网站，出点钱把你心爱的域名买回来。
  • 大陆和大陆以外都有域名贩卖的网站，基本上有以下区别：
    • 大陆：便宜，一个域名大概几块十几块一年，但是购买时需要实名认证（服务器空间在大陆时需要备案）。
    • 以外：稍贵，一年约五六十块，需要填写信息但是不需要实名认证也不需要备案。
  • 大陆反正哪家都要实名我就不推荐了大家随便看着买，大陆以外我在用namesilo，推荐一下。
  • 日常发车选手，安全起见就别在大陆买域名了，不然直接实名了……
  • 关于“我欢天喜地买了个新域名但是该怎么绑定到博客上？！”，以域名贩卖网站的名字为关键词在搜索引擎里搜索，可获得大量教程w

wordpress和hexo，两者在建站方面各有什么优缺点？

wordpress

  • 优点：方便快捷傻瓜式建站，可视化后台界面，适合没基础的人。
  • 缺点：
    • 建成的博客有ip没有域名，要域名需要另外付费。
    • 即使不打算购买域名要用ip裸奔，也需要付费来购买服务器空间（大概每个月二十多块人民币，一年二三百块钱）。
  • 推荐流程BYmooncellwiki群大佬们：在阿里云（轻量？），购买vps（云服务器？），选择大陆以外的服务器（比如香港、新加坡的），根据教程下载软件，架构WordPress / Typecho博客系统。
  • 自搭教程，来自知乎，亲测ok：一键建站系列教程（宝塔面板+wordpress）
  • 发车选手请务必购买大陆以外服务器OTZ

hexo

  • 优点：……建站免费。因为是在github注册帐号，自带了一个二级域名和空间，所以不想出钱的话可以完全免费。
  • 缺点：
    • 属于静态博客，只能在固定的某台电脑上进行推送更新，使用者最好有自己的电脑，有的系统也不支持手机更新博客。
    • 安装过程比较折腾，没有可视化后台，与wordpress相比不够傻瓜。
    • github曾经被墙后来又被放出来了……加载速度稍慢，有一定风险
  • 整体流程：注册github帐号，然后按照博客系统的搭建说明，下载软件和写代码，最后推送至网页。
  • 自搭教程，我就是看着这个搭出来了现在这个博客：GitHub+Hexo 搭建个人网站详细教程
  • github禁止色情内容并且拒绝童车。大概不会管中文这边（毕竟我曾经路过了一篇男性向小黄文），但是请大家尽量尊重一下吧……

个人博客的备案问题？

不要域名只用ip：无论服务器在大陆还是大陆以外都不需要备案。
使用域名且服务器空间在大陆：需要备案。
大陆买的域名，服务器在大陆以外：不用备案。
——以上总结来自mooncellwiki群大佬们

……然而但凡是在大陆网站买的，无论买域名还是买服务器空间，都是要实名制的……开车小伙伴请心里有个数（扭头
